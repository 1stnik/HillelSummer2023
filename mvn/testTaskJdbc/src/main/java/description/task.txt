StudentDatabase : Тестове завдання

Дане тестове завдання повинне в результаті містити працюючу програму, яка задовольняє
всі нижчеописані вимоги, і якісний вихідний код.

Намагайтеся створити грамотну композицію класів і форматувати код відповідно до стандартів
Java.

Опис завдання

Релізувати систему введення та відображення інформації про студентів інституту, що включає
 наступні сутності та їх атрибути:

Student
    first_name,
	last_name,
	date_of_birth,
	email,
	group,
	country

Система повинна мати такі функції:

Відображення списку груп та студентів групи
Відображення списку студентів та фільтрація за першою літерою прізвища
Фільтрування списку студентів за прізвищем та за номером групи
Додавання нового студента, редагування та видалення існуючого
